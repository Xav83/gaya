# Functions

```
include "functions"
```

### `fn.add`

Corresponds to operator `+`.

### `fn.sub`

Corresponds to operator `-`.

### `fn.mult`

Corresponds to operator `*`.

### `fn.div`

Corresponds to operator `/`.

### `fn.less`

Corresponds to operator `<`.

### `fn.greater`

Corresponds to operator `>`.

### `fn.equal`

Corresponds to operator `==`.

### `fn.max`

Return the greater of two objects.

```
@param x <object> The first object.
@param y <object> The second object.
```

### `fn.min`

Return the lesser of two objects.

```
@param x <object> The first object.
@param y <object> The second object.
```
